superstAR


Summer 2023

team: Malaya Heflin, Arti Dhareshwar, Fiza Mulla

tools: Unity, C# script, MadMapper, Arduino

An immersive, "escape-room" experience based primarly on augmented reality interactions. There are 3 sections, each themed around different musical genres: jazz, EDM, and pop. Players travel through each room, solving clues by using a tablet and other technology to interact with their physical environments, helping them earn an instrument from each section. This culminates in a final projection-mapped display where players can see the instruments they've collected along with their winning trophy.

"superstAR" was developed as part of the Immersive Media Design New Works Incubator at the University of Maryland, College Park. It was publicly exhibited on September 7 and 8 at the 2023 NextNow Fest, an annual on-campus arts festival.

✰ my contributions

Technology

I implemented all technological elements used to make superstAR, developing the following features:

  • creating the tablet interactivity and interface within the Unity game engine
  • writing all C# scripts, including AR image tracking and recognition
  • using an Adafruit Circuit Playground with Arduino code to take in and adjust sound input values, which is then used in conjunction with a C# script to produce resulting output in Unity

Art

I created the following art assets:

  • superstAR logo and animation
  • EDM character
  • misc. elements (ex: music note frame)

Other

As a group, we all helped out with the following tasks:

  • arranging props and lighting to make each section more immersive
  • designing and implenting the finale projection-mapped display using MadMapper
  • practicing project management to distribute tasks and create a deadline sheet
  • creating planning documents, including concept art and initial sketches
